# Micrograd-Scratch-


A small educational project that explains **derivatives, backpropagation, and neural networks** from scratch using Python and Jupyter Notebooks.

## Overview

This repository contains notebook-based examples that walk through:

- What derivatives mean
- Why gradients are important in machine learning
- How backpropagation works
- A simple neural network trained with NumPy

## Notebooks

- `notebooks/Micrograd.ipynb`  
  Explains derivatives, slopes, gradients, and the intuition behind backpropagation.

- `notebooks/BackPropagation.ipynb`  
  Implements a simple feedforward neural network with backpropagation using NumPy.

## Features

- Step-by-step math explanations
- NumPy-based neural network implementation
- Forward propagation and backward propagation
- Simple training example on a small dataset

## Requirements

- Python 3.10+
- NumPy
- Matplotlib
- Jupyter Notebook or JupyterLab

## Installation

```bash
pip install -r requirements.txt
```

## How to Run

Start Jupyter Notebook:

```bash
jupyter notebook
```

Then open the notebooks inside the `notebooks/` folder.

## Purpose

This project is meant for learning and understanding the core ideas behind neural networks and backpropagation before moving to larger deep learning frameworks.
